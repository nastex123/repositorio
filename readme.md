# Temperature Converter 1.0

## Description
An interactive Python script that converts temperatures from Celsius to Fahrenheit.  
This project demonstrates user input, variables, arithmetic operations, and formatted output.

## How it works
1. The program welcomes the user and explains its purpose.
2. The user enters a temperature in Celsius using `input()`.
3. The value is converted into a number with `float()`.
4. The formula `(Celsius * 9/5) + 32` is applied to calculate Fahrenheit.
5. The program prints both Celsius and Fahrenheit values, along with a closing message.

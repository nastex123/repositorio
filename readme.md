# Sistema de Inventario Básico

Este proyecto fue hecho por Brandon Carranza y Juan muñoz, e implementamos un programa en Python que permite registrar productos con su nombre, precio y cantidad. El sistema valida las entradas, calcula el costo total y muestra los resultados en consola de manera clara y estructurada.

## Características principales

- Registro de productos con nombre, precio y cantidad.  
- Validación de entradas para asegurar que el precio sea numérico y la cantidad un entero.  
- Cálculo automático del costo total (precio × cantidad).  
- Presentación de resultados en un formato legible.  

## Requisitos

- Python 3.8 o superior.  
- No requiere librerías externas, únicamente la instalación estándar de Python.  

## Uso

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/nastex123/repositorio.git
   ```
2. Acceder al directorio del proyecto:
   ```bash
   cd repositorio
   ```
3. Cambiar a la rama correspondiente:
   ```bash
   git checkout Historia-de-usuario-M1S1
   ```
4. Ejecutar el programa:
   ```bash
   python inventario.py
   ```

## Ejemplo de ejecución

```
Ingrese el nombre del producto: Lápiz
Ingrese el precio del producto: 500
Ingrese la cantidad del producto: 3
Producto: Lápiz | Precio: 500.0 | Cantidad: 3 | Total: 1500.0
```
